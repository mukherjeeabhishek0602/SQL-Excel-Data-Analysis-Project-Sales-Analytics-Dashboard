# SQL-Excel-Data-Analysis-Project-Sales-Analytics-Dashboard
This repository showcases an end-to-end ETL project developed for a business, to analyze customer transactions, sales performance, and business trends. The project demonstrates practical implementation of Data Cleaning, Data Transformation, SQL Querying, Pivot Tables, and Interactive Dashboard Creation using Excel and SQL Server.
The workflow begins with cleaning and preprocessing raw transactional data containing customer details, product information, payment methods, reviews, pricing, and order history from 2006–2020. Data formatting, null handling, date extraction, conditional formatting, and table structuring were performed in Excel. The cleaned dataset was then imported into SQL Server using the Import & Export Wizard for further transformation and analytical querying.
The project includes multiple SQL business analysis queries such as:

Revenue analysis by gender
Payment mode analysis
Top-selling and highest-priced pizzas
Year-wise revenue and order trends
Location-based order distribution

The transformed outputs were used to generate Pivot Tables and an interactive Excel dashboard with multiple visualizations and slicers for dynamic filtering and reporting.
This project highlights skills in:

ETL Process
SQL Querying
Data Cleaning & Transformation
Excel Pivot Tables
Dashboard Development
Business Intelligence & Reporting

The final business insight suggests expanding into home delivery due to increasing online payment adoption and customer demand trends.
